
# MERN Chat App

A Full Stack Chatting App built with the MERN stack.
Uses Socket.io for real-time communication and stores user details in encrypted format in MongoDB.

## Tech Stack

**Client:** React JS
**Server:** Node JS, Express JS
**Database:** MongoDB
  
## Run Locally

Clone the project

```bash
  git clone https://github.com/your-username/mern-chat-app
```

Go to the project directory

```bash
  cd mern-chat-app
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the server

```bash
  npm run start
```

Start the Client

```bash
  // open new terminal
  cd frontend
  npm start
```

# Features

### Authentication
- Secure Login and Signup
- JWT-based authentication
- Guest Login functionality

### Real-Time Chatting
- Instant messaging using WebSockets (Socket.io)
- Typing indicators
- Online/Offline status

### Key Features
- One-to-One chat
- Create Group Chats
- Notifications
- Add or Remove users from groups
- View User Profiles
- AI Chatbot Integration (Xerah AI)
- Dark Mode / Light Mode
- Message Forwarding
- Multi-message Deletion
